ex_video_is_ready

Checks whether a video is ready for playback

Syntax

ex_video_is_ready() Returns true if video is ready for playback, false if not

Description

Checks whether a video is ready for playback. Returns true if yes, false if the video is not ready (connection interrupted, buffering etc.).

Example

//Check if video is still loading and draw a text for it
if (not ex_video_is_ready() and ex_video_is_playing()) {
    draw_text(x, y, "Buffering...");
}

Minimum Supported Version

exVideo 1.1.4
