ex_video_set_background_visibility

Sets the background visibility for the video

Syntax

ex_video_set_background_visibility(visible)
Argument Description
visible The visibility for the video background, true or false
Returns nothing

Description

Sets the visibility of the video background (black area). Sometimes you might want to turn it off so the video appears more seamless with some of your game elements.

Example

//Hide the video background
ex_video_set_background_visibility(false);

Minimum Supported Version

exVideo 1.0.0
