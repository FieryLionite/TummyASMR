ex_video_set_position

Sets the position of the video

Syntax

ex_video_set_position(x, y)
Argument Description
x The x coordinate for the video
y The y coordinate for the video
Returns nothing

Description

Sets the position of a video. Default is 0,0.

Example

//Set the position for the video to 32,32
ex_video_set_position(32, 32);

Minimum Supported Version

exVideo 1.0.0
