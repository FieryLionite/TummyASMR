ex_video_get_background_visibility

Gets the visibility of the video background

Syntax

ex_video_get_background_visibility() Returns the visibility of the video background

Description

Gets the visibility of the video background, 0 to 1.

Example

//Get the visibility of the video background and store it in variable _value
_value = ex_video_get_background_visibility();

Minimum Supported Version

exVideo 1.0.0
