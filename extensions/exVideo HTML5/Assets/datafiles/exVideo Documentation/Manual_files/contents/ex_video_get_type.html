ex_video_get_type

Gets the file type of the video playing

Syntax

ex_video_get_type() Returns the type of the video playing

Description

Gets the video type. Returns "mp4", "m4v", "ogv", "webm" depending on which filetype the browser chose to play (HTML5-only). Will return "embed" if the video is a YouTube embedded video instead (added in exVideo 1.1.2).

Example

//Get the video type
_value = ex_video_get_type();

Minimum Supported Version

exVideo 1.0.0
