ex_video_set_angle3d

Sets the angle of a video in 3D space

Syntax

ex_video_set_angle3d(x, y, z, perspective)
Argument Description
x The X angle in degrees
y The Y angle in degrees
z The Z angle in degrees
perspective The perspective of the 3D space for the rotation
Returns nothing

Description

Sets the angle (rotation) of the video in 3D space. Note in HTML5, not all browsers might support this.

Example

//Rotate the video in 3D space
ex_video_set_angle3d(30, 55, 12, 1800);

Minimum Supported Version

exVideo 1.0.0
