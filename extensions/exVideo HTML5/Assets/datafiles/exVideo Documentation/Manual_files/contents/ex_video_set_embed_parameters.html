ex_video_set_embed_parameters

Sets the embed parameters for the video

Syntax

ex_video_set_embed_parameters(autoplay, controls, autohide, interactive)
Argument Description
autoplay If false, ex_video_play() will only load the video, not play it
controls Whether to show the default video player controls or not
autohide Whether to auto-hide the player controls
interactive Whether to allow user interactivity with the video controls/shortcuts
Returns nothing

Description

Sets several embed parameters of the video (HTML5-only). If autoplay is false, ex_video_play() will only load the video, not play it, this is useful mainly for non-autoplaying YouTube videos. Setting controls to true will force to show the video player default controls. Setting autohide to true will hide the controls after a short time. If interactive is set to false, then all interactivity such as click to pause and keyboard shortcuts will be disabled, useful especially on YouTube videos that require non interrupted playback.

Example

//Embed and show an uninterrupted YouTube video
ex_video_set_embed_parameters(true, false, false, false);
ex_video_play("https://www.youtube.com/watch?v=nxOE21BtpBI", false);

Minimum Supported Version

exVideo 1.1.2
