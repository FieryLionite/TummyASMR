ex_video_is_playing

Checks whether a video is playing

Syntax

ex_video_is_playing() Returns true if video is playing, false if not

Description

Checks whether a video is playing. Returns true if playing, false if not.

Example

//Check if video "test.mp4" is playing and if not, play it
if (!ex_video_is_playing()) {
    ex_video_play("test.mp4", false);
}

Minimum Supported Version

exVideo 1.0.0
