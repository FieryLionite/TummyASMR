ex_video_is_paused

Checks whether a video is paused

Syntax

ex_video_is_paused() Returns true if video is paused, false if not

Description

Checks whether a video is paused. Returns true if paused, false if not.

Example

//Check if video is paused and if yes, resume it
if (ex_video_is_paused()) {
    ex_video_resume();
}

Minimum Supported Version

exVideo 1.0.0
