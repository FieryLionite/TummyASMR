ex_video_set_size

Sets the width and height of the video

Syntax

ex_video_set_size(width, height)
Argument Description
width The width for the video
height The height for the video
Returns nothing

Description

Sets the width and height of a video. Default is the game's dimensions.

Example

//Set the dimensions for the video to 320, 240
ex_video_set_size(320, 240);

Minimum Supported Version

exVideo 1.0.0
