ex_video_set_track_position

Sets the current track position of the video

Syntax

ex_video_set_track_position(position)
Argument Description
position The track position, in seconds
Returns nothing

Description

Sets the track position of the video in seconds, e.g. 35.12

Example

//Set the track position to 2 seconds
ex_video_set_track_position(2);

Minimum Supported Version

exVideo 1.0.0
